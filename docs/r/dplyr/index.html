    レコード数のカウント

    以下の3つは全て同じ結果を返す。カラム x の値ごとにレコード数をカウントする。

    df %>%
  group_by(x) %>%
  summarize(n = n())

df %>%
  group_by(x) %>%
  tally()

df %>%
  count(x)

    add_count() と add_tally() は、上の例の summarize() を mutate() に変えたもの、元の df の行数は変えずにグループごとのカウント列を追加する

    df %>%
  group_by(x) %>%
  mutate(n = n())

df %>%
  group_by(x) %>%
  add_tally()

df %>%
  add_count(x)
